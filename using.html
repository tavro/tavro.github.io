using ⚒️

this page was inspired by Stefan VanBuren and Shreyans Devendra Doshi, so thanks to them! here, i'll list the tools i use on a daily basis. i'll do my best to keep this list updated and might eventually throw in some fun extras that i use regularly.

software

category home work comment
os linux (desktop & laptop), android (phone) linux (desktop), macOS (laptop) i always run the latest stable version of ubuntu on my desktop linux machines, and i use the latest kali release on my personal laptop.
editor emacs android studio, vscode i've been curious about switching to vim, since many of my friends use it. but i learned to use emacs in university, and my procedural memory has kept me from making the switch. i'm currently building my own editor, so i'll probably end up using that, if it ever gets finished that is...!
browser opera, firefox (desktop & phone) i like the features opera offers, but i prefer firefox's developer tools (or maybe i'm just more familiar with them).
music player spotify not much to say, unified streaming choice across devices
terminal GNOME terminal consistent terminal environment across linux-based systems, although, i'm interested in building my own terminal as well... we'll see if that ever happens.
communication discord (desktop), SMS slack, gmail for work and organizations i am using slack or gmail. since i have a dumb phone i currently only use SMS on my personal phone.

hardware

category home work description
stationary computer custom-built desktop computers i will get rid of my desktop computer at work soon enough, so i won't dig into the specs because of that. i might list my personal desktop computer specs soon
laptop crappy lenovo thinkpad macbook m4 pro i use a macbook for work, mainly to run xcode and build for ios. my lenovo is used for ethical hacking stuff
keyboard cmyk 87/108 ducky origin vintage i'm planning to upgrade to a kinesis ergonomic keyboard for work. my colleuge has one, and it seems super nice.
mouse logitech mx vertical lenovo go wireless vertical i really like the logitech mouse, the lenovo one is too small for my hands and i will probably replace it very soon, although i try not to use my mouse too much.
phone motorola g05 (with dumb phone launcher) n/a i was interested in buying a true dumb phone, like the light phone 2 or something similar, but they're still too expensive, unavailable where i live or too buggy. plus, i needed something that could run spotify and banking apps. i ended up buying my motorola for less than $90.
speakers mackie cr n/a i bought them back in highschool and they are still getting the job done
microphone shure sm 7 b + behringer xenyx q502 sound interface n/a when i make discord calls, i use this microphone. besides that, i sometimes record screencasts and similar content. every now and then, i also record vocals for songs i produce.
headphones sonos ace i would describe them as a lightweight android alternative to airpods max. i really love them and don't see myself replacing them anytime soon.
camera canon eos 500 (film), nikon coolpix p510 (digital), IMX789 sensor fine-tuned by hasselblad (casual) n/a for film photography, i'm using a canon eos 500 that i found in a second-hand store. the nikon coolpix p510 is my main camera, it has a really nice optical zoom, which you won't find in any smartphone, for example. i like this because you can take good photos from afar, and for people interested in birds, like me, it's a great feature. for more casual photos, i use a oneplus phone dedicated solely to photography.
note taking remarkable 2 tablet, paperblanks (physical) for journaling, sketching or quick notes
clock casio f-91w digital wrist clock, it's nice for keeping track of time! although it's currently a few minutes behind, i use that as a feature, haha. i like to be on time.
